Object concept in oriented object programming

Definition : Object

An object is an instance of a class.

Characteristics of an object

An object is defined by:

• a name: the name of the object

• one or more states (x=0, y=-1)

• one or more behaviors (treatments)

Fundamental : Static instanciation

Static instanciation consist in build an object by calling a class constructor implicitely.

This object is loaded in the stack.

Fundamental : Dynamic instanciation
Dynamic instanciation consist in build an object by calling a constructor explicitely with the new operator.
This created object is a pointer loaded in the RAM.
To destruct the object, the destructor should be called explicitely with the delete operator.
1
#include "point2D.hpp"
2
3
int main()
4
{
5
   //Static instanciation
6
   Point2D P1(-1,-1);
7
   P1.display();
8
9
   //Dynamic instanciation
10
   Point2D *P2 = new Point2D(1,1);
11
   P2->display();
12
   delete(P2);
13
}
#include "point2D.hpp"

int main()
{
   //Static instanciation
   Point2D P1(-1,-1);
   P1.display();

   //Dynamic instanciation
   Point2D *P2 = new Point2D(1,1);
   P2->display();
   delete(P2);
}
